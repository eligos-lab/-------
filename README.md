Courier Route Optimizer (Java + Swing + ORS + интерактивная карта)

Настольное кроссплатформенное приложение для курьера: принимает список адресов доставки, оптимизирует порядок посещения (минимизация времени/километража на основе матрицы), отображает маршрут на интерактивной карте и позволяет экспортировать результат в CSV / PDF.

Возможности

Ввод стартового адреса (опционально) и списка адресов доставок (1 адрес = 1 строка)

Геокодирование адресов через OpenRouteService (ORS)

Построение матрицы расстояний/времени и оптимизация порядка посещения:

базовый greedy (nearest neighbor)

улучшение маршрута алгоритмом 2-opt

Отображение маршрута на интерактивной карте (Leaflet + OpenStreetMap) внутри Swing через JavaFX WebView
